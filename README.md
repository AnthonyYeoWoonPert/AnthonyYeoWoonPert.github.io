# AnthonyYeoWoonPert.github.io

A calm, mobile-first portfolio site for Anthony Yeo Woon Pert, highlighting full-stack software engineering experience with an emphasis on clarity, readability, and story-driven content. Built as a lightweight static site to present experience, projects, skills, and leadership in a professional format aligned with a .NET/C#/React/MSSQL background.

## What it demonstrates
- Semantic HTML structure with clear, scannable hierarchy for professional storytelling.
- Responsive, mobile-first layout with large typography and generous spacing.
- Accessible navigation and focus states for keyboard users.
- Simple, maintainable styling without heavy frameworks.

## Tech stack
- HTML5
- CSS3
- JavaScript (minimal)
- Bootstrap + jQuery (included assets)

## How to run
- Open `index.html` directly in a browser, or
- Serve locally:
  ```bash
  python -m http.server 8000
  ```
  Then visit `http://localhost:8000`.
